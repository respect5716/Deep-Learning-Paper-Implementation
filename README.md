# Deep-Learning-Paper-Implementation



## 목적

* 딥러닝 논문 구현



## 환경

* Google Colaboratory Pro
* GPU : Tesla P100
* Framework : Tensorflow 2.x



## 주의 사항

* 개인의 연구와 학습을 목적으로 하기 때문에 데이터셋과 하이퍼파라미터가 원본과 상이할 수 있음
